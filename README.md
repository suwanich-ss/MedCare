
# Elderly Medication Tracking Application🚀

MedCare เป็นแอปพลิเคชันสำหรับช่วยติดตามและลดความผิดพลาดในการรับประทานยาของผู้สูงอายุ โดยออกแบบมาเพื่อแก้ปัญหาการลืมรับประทานยา การรับประทานยาผิดเวลา และช่วยให้ลูกหลานหรือผู้ดูแลสามารถติดตามสถานะการรับประทานยาของผู้สูงอายุได้จากระยะไกลผ่านระบบดิจิทัล

ระบบถูกออกแบบภายใต้แนวคิด Elder-Centered Design เพื่อให้ผู้สูงอายุสามารถใช้งานได้ง่าย ผ่านปุ่มขนาดใหญ่ ตัวอักษรขนาดใหญ่ ระบบอ่านออกเสียงภาษาไทย และขั้นตอนการใช้งานที่ไม่ซับซ้อน

## Features

- For Elderly Users
    -
    - แจ้งเตือนเวลารับประทานยา
    - แสดงรูปภาพยาและข้อมูลยา
    - อ่านออกเสียงชื่อยาและวิธีรับประทานยา
    - ยืนยันการรับประทานยาด้วยปุ่ม "กินยาแล้ว"
    - ดูประวัติการรับประทานยา
    - รองรับการใช้งานแบบ Offline
- Family Members
    -
    - เพิ่มและจัดการข้อมูลยา
    - ตั้งเวลารับประทานยา
    - ติดตามสถานะการรับประทานยาแบบ Real-time
    - รับการแจ้งเตือนเมื่อผู้สูงอายุลืมรับประทานยา
    - ดูประวัติการรับประทานยาย้อนหลัง
- OCR System
    -
    - สแกนฉลากยา
    - อ่านข้อความภาษาไทยและภาษาอังกฤษจากฉลากยา
    - ลดภาระในการกรอกข้อมูลด้วยตนเอง
    - แสดงผลให้ผู้ใช้งานตรวจสอบก่อนบันทึก
